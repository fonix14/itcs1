1) Добавить импорты в app/main.py:

from app.api.command_center import router as command_center_router
from app.ui_command_center import router as ui_command_center_router

2) Подключить роутеры после существующих include_router(...):

app.include_router(command_center_router)
app.include_router(ui_command_center_router)

3) После деплоя проверить:

curl -s http://localhost:8080/api/command-center/overview | python -m json.tool
curl -s -o /dev/null -w "%{http_code}\n" http://localhost:8080/ui/command-center
